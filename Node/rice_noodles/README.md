# 米线

#### 介绍
此项目是用express框架开发的mvc米线外卖系统，欢迎订购米线，很好吃呀！

一、前端知识体系
1、HTML5
2、CSS3
3、JAVASCRIPT（nodejs+express框架+YDUI框架）

二、数据库知识体系
MongoDB

三、开发工具
webstorm

四、项目管理体系
GIT（码云）

四、架构模式
+-- Project Folder
    +-- app
    |   \-- controllers——index、admin
    |   \-- routes
    |   \-- view——index、admin
    |
    +-- public
    |   \-- index——css、img
    |   \-- admin——css、img
